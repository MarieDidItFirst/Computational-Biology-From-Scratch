🎀UCSF ChimeraX is a visualization and analysis tool used for molecular structures.

🧬Topics Covered
- Installation
- User interface
- Structure visualization
- Surface generation
- Distance measurements
- Hydrogen bond analysis
- Structural comparison
- Preparing structures for docking

✔️Learning Objectives
By the end of this section, you should be able to:
- Visualize proteins and ligands
- Perform basic structural analyses
- Prepare structures for downstream applications
