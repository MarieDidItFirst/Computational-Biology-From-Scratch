🎀Computational biology uses many different file formats. Understanding their purpose is essential before working with biological data and software tools.

🖥️Topics Covered
- FASTA
- PDB (Legacy vs CIF Format)
- PDBQT
- MOL2
- SDF
- CSV
- TXT
- GRO
- ITP
- SMILES
- CXS

✔️Learning Goal
By the end of this section, you should understand what each file format contains, where it is used, and how it fits into computational biology workflows.
