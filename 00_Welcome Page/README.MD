Welcome to ✨Computational Biology✨ from Scratch.
This repository is a beginner-friendly collection of notes, guides, tutorials, work-flows, code snipepts, troubleshooting guides, and project examples designed for students learning computational biology independantly.

I created this repository because when I started learning computational biology, I often felt lost, unable to find resources that explained not only what to do, but also why each step is necessary. Many tutorials assumed prior knowledge and I did'nt even know basics of coding.So my entire learning process was very confusing and frustrating.

The goal of this repository is to provide clear practical, and accesible guidance for students working with tools like PyMOL, Chimera, AutoDock, AutoDock Vina, GROMACS, R, Python, and cancer bioinformatics platforms.
